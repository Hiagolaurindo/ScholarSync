# Documento de Visão — ScholarSync

## 1. Introdução

Este documento apresenta a visão geral do sistema ScholarSync, desenvolvido para auxiliar estudantes na organização de atividades acadêmicas.

O sistema busca facilitar o gerenciamento de disciplinas, tarefas e prazos, permitindo melhor controle da rotina universitária.

---

## 2. Objetivo do Sistema

O objetivo do ScholarSync é fornecer uma plataforma simples e eficiente para organização acadêmica, permitindo que estudantes acompanhem suas atividades e prazos em um único ambiente.

---

## 3. Problema Identificado

Muitos estudantes possuem dificuldades para organizar tarefas, trabalhos e datas importantes da faculdade, utilizando anotações dispersas ou ferramentas pouco práticas.

Isso pode causar:

- Perda de prazos
- Falta de organização
- Dificuldade no acompanhamento das atividades
- Baixa produtividade acadêmica

---

## 4. Solução Proposta

O ScholarSync centraliza as informações acadêmicas em uma única plataforma, permitindo:

- Cadastro de disciplinas
- Criação de tarefas
- Organização de atividades
- Controle de prazos
- Gerenciamento das informações acadêmicas

---

## 5. Público-Alvo

- Estudantes universitários
- Alunos de cursos técnicos
- Usuários que desejam organizar tarefas acadêmicas

---

---

## 8. Histórias de Usuário

### HU01 — Cadastro de usuário

Como estudante, quero criar uma conta para acessar o sistema.

### HU02 — Cadastro de disciplinas

Como estudante, quero cadastrar disciplinas para organizar minhas matérias.

### HU03 — Cadastro de tarefas

Como estudante, quero cadastrar tarefas com prazo para controlar minhas atividades acadêmicas.

### HU04 — Gerenciamento de tarefas

Como estudante, quero editar ou excluir tarefas para manter minhas informações atualizadas.
